# Capacitor Docs

官方 [Capacitor](https://capacitorjs.com) 文档，使用 [Docusaurus](https://docusaurus.io) 构建。基于 [Ionic Docs](https://github.com/ionic-team/ionic-docs)

---

## 开始使用

安装 [Node 和 npm 的最新版本](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)。

安装 npm 后，运行

```
npm install
npm run build
npm run serve
```

站点将以开发模式启动。
