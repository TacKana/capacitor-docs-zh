# Capacitor 文档

官方 [Capacitor](https://capacitorjs.com) 文档，使用 [Docusaurus](https://docusaurus.io/) 构建。基于 [Ionic 文档](https://github.com/ionic-team/ionic-docs) 开发。

---

## 快速开始

请安装 [最新版本的 Node 和 npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)。

安装完成后，运行以下命令：

```
npm install
npm run build
npm run serve
```

网站将以开发模式启动。
