## 社区维护

Capacitor 自诞生之日起就保持 100% 开源（MIT 许可），并将一直延续这一传统。通过社区协作的维护策略，开发者可以确信 Capacitor 是其跨平台应用开发的正确选择。Capacitor 团队定期发布新版本和错误修复，并热忱欢迎社区提交拉取请求。

## 版本维护与支持状态

考虑到时间和资源的现实限制，以及需要持续适配移动平台的快速演进，Capacitor 团队会适时发布新的主版本。我们将竭尽所能确保版本升级过程的顺畅。团队建议开发者始终升级到最新版 Capacitor，以获取最新功能、改进和稳定性更新。

各版本当前状态如下：

| 版本 |      状态      |    发布日期    |   维护终止   | 扩展支持终止 |
| :--: | :------------: | :------------: | :----------: | :----------: |
|  V7  |    **维护中**  | 2025年1月20日 |    待定     |    待定     |
|  V6  |  扩展支持阶段  | 2024年4月15日 | 2025年7月20日 | 2026年1月20日 |
|  V5  |  终止支持  |  2023年5月3日  | 2024年10月15日 |  2025年4月15日 |
|  V4  |  终止支持  |  2022年7月27日 | 2023年11月3日 |  2024年5月3日 |
|  V3  |  终止支持  |  2021年5月18日 | 2023年1月27日 |  2023年7月27日 |
|  V2  |  终止支持  |  2020年4月3日 | 2021年11月18日 |  2022年5月18日 |
|  V1  |  终止支持  |  2019年5月23日 | 2020年10月3日 |  2021年4月3日 |

- **维护阶段**：仅修复重大错误和安全问题，不添加主要功能改进
- **扩展支持**：不再向新用户提供
