[WebNative Visual Studio Code 扩展](https://marketplace.visualstudio.com/items?itemName=webnative.webnative)是一个由社区维护的插件，帮助你执行常见的 Capacitor 开发任务，无需记忆 CLI 命令。

如果你在这台电脑上安装了 VS Code，请点击下方的 Install 按钮。你也可以通过搜索 "WebNative" 找到该扩展。
