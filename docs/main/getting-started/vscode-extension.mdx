[WebNative Visual Studio Code 扩展](https://marketplace.visualstudio.com/items?itemName=webnative.webnative) 是一个由社区维护的插件，它能帮助你执行常见的 Capacitor 开发任务，而无需记忆 CLI 命令。

如果你的电脑上安装了 VS Code，请点击下方的“安装”按钮。你也可以通过搜索“WebNative”来找到此扩展。
