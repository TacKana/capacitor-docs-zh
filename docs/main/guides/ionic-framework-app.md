# 构建 Ionic 相册应用

**支持平台**：Web、iOS、Android

Capacitor 能够轻松构建可在 iOS、Android、桌面和网页端原生运行的 Web 应用。本教程将指导您构建一个全平台兼容的完整相册应用。

> 本教程最新维护版本已迁移至 Ionic 官方文档站，[最新版](https://ionicframework.com/docs/intro/next)包含 Angular、React 和 Vue 的实现教程。
