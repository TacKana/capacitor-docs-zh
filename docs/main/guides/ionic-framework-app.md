# 构建 Ionic 照片库应用

**平台**: 网页、iOS、Android

Capacitor 让构建能在 iOS、Android、桌面端和网页上原生运行的 Web 应用变得简单。在本指南中，我们将构建一个能在所有平台上运行的完整照片库应用。

> 本指南现已在 Ionic 文档中维护。[最新版本](https://ionicframework.com/docs/intro/next) 包含针对 Angular、React 和 Vue 的教程。
