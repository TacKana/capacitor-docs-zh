# Capacitor 社区插件

社区已经构建了许多 Capacitor 插件，以增强你的应用功能。

有多种方式可以找到社区插件：你可以搜索网络或 npm 来查找任何已发布的插件，或者浏览官方的 [Capacitor Community](https://github.com/capacitor-community) GitHub 组织和 [npm 作用域](https://npmjs.com/~capacitor-community)，那里有一份精心挑选的、高质量的、由社区支持的 Capacitor 插件列表。

## Capacitor Community

官方的 [Capacitor Community](https://github.com/capacitor-community) GitHub 组织和 [npm 作用域](https://npmjs.com/~capacitor-community) 是一项新举措，旨在汇集最好的、由社区支持的 Capacitor 插件，并提供一致的项目和代码规范。Capacitor 核心团队会协助相关工作，但并不官方维护该组织中的任何插件。

[探索 Capacitor Community &#8250;](https://github.com/capacitor-community)

## Capawesome

[Capawesome](https://github.com/capawesome-team) GitHub 组织是我们的另一位 Ionic 开发者专家发起的社区项目，目标是为整个社区提供出色的 Capacitor 插件。
其中包括 [capacitor-firebase](https://github.com/capawesome-team/capacitor-firebase) 和 [capacitor-mlkit](https://github.com/capawesome-team/capacitor-mlkit) 等。
请关注 [Capawesome 博客](https://capawesome.io/blog/) 或 [Twitter](https://twitter.com/capawesomeio) 以获取最新动态。

## Cordova 生态系统

Capacitor 支持大多数 Cordova 插件，因此开发者可以在他们的 Capacitor 应用中使用 [数百个现有的 Cordova 插件](https://cordova.apache.org/plugins/)。虽然某些 Cordova 插件与 Capacitor [不兼容](/plugins/cordova.md#已知不兼容的插件)，但大多数是兼容的。所以，如果没有现成的 Capacitor 专用插件，尝试使用 Cordova 插件是值得的。

另外，你可以查看 [`@awesome-cordova-plugins` 社区](https://github.com/danielsogl/awesome-cordova-plugins) 插件列表，其中主要是 Cordova 插件，并提供了便捷的包装器，使它们更容易在你的应用中使用。你可以随意使用 `@awesome-cordova-plugins` 社区的包装器，或者仅将该列表作为高质量 Cordova 插件的参考。
