# Capacitor 企业版

Capacitor 为关键任务型企业级应用提供强大的原生安全、身份验证和数据管理解决方案与插件。

此外，团队还能获得深受客户喜爱的技术支持服务，以及可选的专业咨询服务，以确保您的下一个重要应用项目取得成功并保持领先，无论移动生态系统如何变化。

[Capacitor 企业版方案](/plugins/enterprise.md) 围绕一套强大的原生 Capacitor 插件构建：[Auth Connect](https://ionicframework.com/enterprise/auth-connect) 用于安全身份验证，[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 用于加密会话和用户身份管理，以及 [Offline Storage](https://ionicframework.com/enterprise/offline-storage) 用于高性能、加密的 SQL 数据存储。

每个解决方案都能与在 Capacitor 中运行的任何现代 Web 技术栈完美配合，并且不需要 Ionic Framework。

## 企业级身份验证

借助 [Auth Connect](https://ionicframework.com/enterprise/auth-connect) 和 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，您可以使用移动安全最佳实践（如生物识别身份验证和加密会话存储）连接到任何 OAuth 或 OpenID 提供商（如 Auth0、Ping、AWS Cognito、Azure Active Directory 等）。

Auth Connect 为众多企业和消费者身份提供商提供了一个简单、可移植的身份验证 API，只需几行 JavaScript 代码即可实现复杂的原生身份验证流程。

结合 Identity Vault（见下文），Capacitor 应用能够实现最强大、最安全且最“正确”的企业级身份验证流程。

## 安全的身份管理

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 与 Auth Connect 协同工作，以保护敏感的用户和身份数据（如身份验证令牌），其背后由支持最广泛 iOS 和 Android 设备的生物识别身份验证提供支持。Identity Vault 超越了指纹或面部扫描插件的范畴，能够与各平台上的安全区域和加密服务协同工作。这是必要的，因为像大多数社区插件那样仅扫描指纹或面部身份，并不能提供足够的安全性，也无法有效防御越狱或其他攻击。

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 也可以独立使用，并与任何现有的身份验证流程集成，以提供高级的会话安全性。

## 离线与加密数据存储

借助 [Offline Storage](https://ionicframework.com/enterprise/offline-storage)，Capacitor 开发者可以获得一个高性能、加密的 SQL 数据存储系统，用于在 iOS 和 Android 上构建功能强大且安全的离线应用。

结合 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，开发者可以安全地管理数据加密密钥，并实现设备端数据管理的最佳实践，而无需编写一行原生代码。
