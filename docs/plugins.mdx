# Capacitor 插件

Capacitor 中的插件让 JavaScript 能够直接与原生 API 交互。

  </DocCard>

<br />

Web 应用可以通过插件访问原生 API 的全部能力。插件封装了常见的原生操作，这些操作在不同平台上可能使用截然不同的 API，但通过插件对外暴露一致的跨平台 JavaScript API。

此外，Capacitor 的插件机制使得传统原生开发者和 Web 开发者混合的团队能够协作开发应用的不同部分。

Capacitor 会自动在客户端生成 JavaScript 钩子，因此大多数插件只需使用 Swift/Obj-C（iOS）和/或 Java/Kotlin（Android）编写原生代码。当然，也可以为插件添加自定义的 JavaScript。
