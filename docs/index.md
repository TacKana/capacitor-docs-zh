# Capacitor：面向 Web 应用的原生跨平台运行时

Capacitor 是一款原生跨平台运行时，它能让你轻松构建高性能的移动应用，这些应用使用现代 Web 工具链，在 iOS、Android 等平台上原生运行。作为混合应用的下一次演进，Capacitor 创造了 **Web 原生应用**，为那些希望以 Web 优先方式构建应用、同时在需要时又能完全访问原生 SDK 的团队，提供了一种现代化的原生容器方案。

![Capacitor 徽标与背景](/img/v6/docs/capacitor-index.png)

## 介绍

Capacitor 提供了一套一致、以 Web 为核心的 API，使应用能够尽可能贴近 Web 标准，同时在支持的平台上访问丰富的原生设备功能。如果某个功能在浏览器中能运行，那么在使用 Capacitor 的移动应用中很可能也能运行。通过为 iOS 的 Swift、Android 的 Java 以及 Web 的 JavaScript 提供的插件 API，添加原生功能变得简单直接。

## 快速开始

Capacitor 上手非常容易！它可以轻松集成到任何现有的现代 JavaScript 项目中，或者从头创建一个全新的 Capacitor 项目。

请按照 [安装指南](/main/getting-started/installation.md) 开始构建你的应用。
