# Capacitor：跨平台原生运行时，用于 Web 应用

Capacitor 是一个跨平台原生运行时，利用现代 Web 工具，让构建在 iOS、Android 及更多平台上原生运行的高性能移动应用变得简单。作为混合应用的下一个演进，Capacitor 创建了 **Web 原生应用**，为那些希望以 Web 为先同时在不牺牲对原生 SDK 完全访问权限的团队，提供了一种现代化的原生容器方案。

![Capacitor Logo and background](/img/v6/docs/capacitor-index.png)

## 介绍

Capacitor 提供了一套一致的、以 Web 为中心的 API，使应用能够尽可能贴近 Web 标准，同时在支持它们的平台上访问丰富的原生设备功能。如果在浏览器中能运行，那么使用 Capacitor 后在移动应用中也大概率能运行。通过基于 Swift（iOS）、Java（Android）和 JavaScript（Web）的插件 API，添加原生功能非常简单直接。

## 开始使用

开始使用 Capacitor 非常简单！Capacitor 可以接入任何现有的现代 JavaScript 项目，也可以从头创建一个全新的 Capacitor 项目。

请遵循[安装指南](/main/getting-started/installation.md)开始构建你的应用。
