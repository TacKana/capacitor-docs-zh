# Capacitor 企业版

Capacitor 为关键任务型企业应用提供强大的原生安全、认证和数据管理解决方案及插件。

此外，团队还可以获得深受客户喜爱的支持和可选的咨询服务，以确保您的下一个主要应用项目取得成功，并且无论移动生态系统如何变化，都能保持稳定运行。

[Capacitor 的企业版产品](/plugins/enterprise.md) 围绕一套强大的原生 Capacitor 插件构建：[Auth Connect](https://ionicframework.com/enterprise/auth-connect) 用于安全认证，[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 用于加密会话和用户身份管理，以及 [Offline Storage](https://ionicframework.com/enterprise/offline-storage) 用于高性能加密 SQL 数据存储。

每个解决方案都能很好地与任何在 Capacitor 上运行的现代 Web 技术栈配合使用，并且不需要 Ionic Framework。

## 企业级认证

通过 [Auth Connect](https://ionicframework.com/enterprise/auth-connect) 和 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，可以连接到任何 OAuth 或 OpenID 提供商（Auth0、Ping、AWS Cognito、Azure Active Directory 等），并使用移动安全最佳实践，如生物识别认证和加密会话存储。

Auth Connect 提供了一个简单、可移植的 API，用于与众多企业和消费者身份提供商进行认证，只需几行 JavaScript 代码即可实现复杂的原生认证流程。

结合 Identity Vault（见下文），Capacitor 应用可以实现最强大、最安全且_正确_的企业认证流程。

## 安全身份管理

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 与 Auth Connect 协同工作，保护敏感的用户和身份数据（如认证令牌），并由生物识别认证提供支持，兼容最广泛的 iOS 和 Android 设备。Identity Vault 超越了典型的指纹或面部扫描插件，与每个平台上的安全飞地和加密服务协同工作。这一点至关重要，因为像大多数社区插件那样仅扫描指纹或面部身份，并不能提供足够的安全性，也无法防止越狱或其他攻击。

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 也可以独立使用，并与任何现有的认证流程集成，以提供高级的会话安全保护。

## 离线与加密数据存储

借助 [Offline Storage](https://ionicframework.com/enterprise/offline-storage)，Capacitor 开发者可以访问一个高性能、加密的 SQL 数据存储系统，用于在 iOS 和 Android 上构建功能强大且安全的离线应用。

结合 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，开发者无需编写一行原生代码即可安全地管理数据加密密钥，并实现设备端数据管理最佳实践。
