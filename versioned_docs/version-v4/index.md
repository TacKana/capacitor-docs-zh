# Capacitor：面向 Web 应用的跨平台原生运行时

Capacitor 是一个跨平台原生运行时，它让开发者能够轻松构建高性能的移动应用，这些应用使用现代 Web 工具链，并能在 iOS、Android 等平台上原生运行。作为混合应用演进的下一阶段，Capacitor 创造了 **Web 原生应用（Web Native apps）**，为希望在保持 Web 优先开发模式的同时，又不牺牲对原生 SDK 完全访问权的团队，提供了一种现代化的原生容器解决方案。

![Capacitor Logo and background](/img/v4/docs/capacitor-index.png)

## 介绍

Capacitor 提供了一套专注于 Web 的一致性 API，使应用能够尽可能贴近 Web 标准，同时又能访问支持平台上丰富的原生设备功能。如果一个功能在浏览器中能运行，那么在使用 Capacitor 的移动应用中很可能也能运行。通过为 iOS 的 Swift、Android 的 Java 以及 Web 的 JavaScript 提供的插件 API，添加原生功能变得简单直接。

## 快速开始

开始使用 Capacitor 非常简单！Capacitor 可以集成到任何现有的现代 JavaScript 项目中，也可以从头创建一个全新的 Capacitor 项目。

请按照 [安装指南](/main/getting-started/installation.md) 开始构建您的应用。
