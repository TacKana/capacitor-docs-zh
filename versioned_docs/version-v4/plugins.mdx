# Capacitor插件

Capacitor中的插件让JavaScript能够直接调用原生API接口。

  </DocCard>

<br />

通过插件，Web应用能够充分利用原生API的全部能力。插件封装了那些在不同平台上API差异较大的原生操作，同时为JavaScript提供统一的跨平台接口。

此外，Capacitor的插件功能使得传统原生开发者与Web开发者可以协同工作，分别负责应用的不同模块。

Capacitor会自动在客户端生成JavaScript钩子，因此大多数插件只需为iOS平台编写Swift/Obj-C代码，和/或为Android平台编写Java/Kotlin代码。当然，您也可以为插件添加自定义的JavaScript逻辑。
