# Capacitor 插件

Capacitor 中的插件使 JavaScript 能够直接与原生 API 进行交互。

<br />

借助插件，Web 应用可以访问原生 API 的全部功能。插件封装了常见的原生操作，这些操作在不同平台上可能使用完全不同的 API，同时向 JavaScript 暴露一致的、跨平台的 API。

此外，Capacitor 的插件功能使得由传统原生开发者和 Web 开发者组成的团队能够协同工作，共同开发应用的不同部分。

Capacitor 会自动在客户端生成 JavaScript 钩子，因此大多数插件只需要为 iOS 使用 Swift/Obj-C 和/或为 Android 使用 Java/Kotlin。当然，也可以为插件添加自定义的 JavaScript。
