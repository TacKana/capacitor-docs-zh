按照以下步骤为 Angular、React 或 Vue 创建一个**新的初始项目**。

<video style={{borderRadius:5}} width="100%" controls>
      <source src={require('@site/static/video/new-project.mp4').default} type="video/mp4" />
</video>

1.  为你的项目创建一个文件夹，并在 VS Code 中打开它
2.  点击侧边菜单上的 `Ionic` 徽标
3.  选择你偏好的框架
4.  从各种初始项目模板中选择
5.  输入项目的 `name`，然后按 `Enter` 键

经过短暂的安装过程后，你的初始项目就准备好可以开始开发了。

无论使用何种框架，你也可以[将现有的 Web 应用程序转换为原生应用](web-to-native)。
