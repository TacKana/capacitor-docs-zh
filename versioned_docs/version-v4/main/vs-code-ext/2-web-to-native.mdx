你可以使用 Capacitor 将**任何** Web 应用或网站转换为原生应用（通常只需几分钟）。

<video style={{borderRadius:5}} width="100%" controls>
      <source src={require('@site/static/video/existing-project.mp4').default} type="video/mp4" />
</video>

1. 点击侧边菜单栏的 `Ionic` 图标
2. 选择 `Add Capacitor Integration`（添加 Capacitor 集成）
3. 选择 `Add iOS Project`（添加 iOS 项目）来添加原生的 **XCode** 项目
4. 选择 `Add Android Project`（添加 Android 项目）来添加原生的 **Android Studio** 项目
