# 构建 Ionic 照片库应用

**平台**: Web, iOS, Android

Capacitor 使得构建能够在 iOS、Android、桌面端和 Web 上原生运行的 Web 应用变得简单。在本指南中，我们将构建一个在所有平台上都能运行的完整照片库应用。

> 本指南现已在 Ionic 官方文档中维护。[最新版本](https://ionicframework.com/docs/intro/next) 提供了针对 Angular、React 和 Vue 的教程。
