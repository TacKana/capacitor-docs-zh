# 构建 Ionic 相册应用

**支持平台**：网页、iOS、Android

Capacitor 能轻松构建可原生运行于 iOS、Android、桌面端和网页端的应用程序。本教程将带您完成一个全平台兼容的相册应用开发。

> 本指南最新维护版本已迁移至 Ionic 官方文档，[最新版教程](https://ionicframework.com/docs/intro/next) 包含 Angular、React 和 Vue 的实现方案。
