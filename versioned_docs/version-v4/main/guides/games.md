# 使用 Capacitor 进行游戏开发

Capacitor 是构建跨平台游戏的绝佳平台。凭借对 WebGL 和 Canvas 渲染的广泛支持，开发者能够在现代移动设备上构建高性能的游戏体验。

我们通常建议使用游戏引擎进行开发，它们为常见的游戏对象和功能提供了基础组件，当然开发者也可以选择从零开始构建。

## Phaser + Capacitor

[Phaser](https://phaser.io/) 是最流行的基于 Web 的游戏开发引擎之一。

要开始使用 Capacitor 和 Phaser 构建游戏，请查看 [Capacitor + Phaser 入门模板](https://github.com/ionic-team/capacitor-starters/tree/main/phaser)，该模板已考虑了移动设备缩放问题，并默认设置为横屏模式。
