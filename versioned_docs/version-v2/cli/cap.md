# Capacitor CLI - cap 命令

查看所有可用的 CLI 命令及选项。

```bash
npx cap [-V] [-h]
```

<strong>选项参数：</strong>

- `-V, --version` (可选)：显示版本号
- `-h, --help` (可选)：显示帮助信息。该选项也可用于单个命令查询。
