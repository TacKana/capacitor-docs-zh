# Capacitor CLI - cap

查看所有可用的 CLI 命令和选项。

```bash
npx cap [-V] [-h]
```

<strong>选项：</strong>

- `-V, --version`（可选）：输出版本号
- `-h, --help`（可选）：输出使用信息。也可与单独的命令一起使用。
