# 插件生成

创建一个新的自定义 Capacitor 插件。这将启动一个向导，提示您输入有关新插件的信息。更多关于开发插件的信息请参见 [此处](/plugins.md)。

```bash
# 项目中已安装 Capacitor CLI
npx cap plugin:generate

# 未安装 Capacitor CLI
npx @capacitor/cli@2.4.7 plugin:generate
```
