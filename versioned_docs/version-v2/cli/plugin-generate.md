# 插件生成

创建一个新的自定义 Capacitor 插件。此命令将启动一个向导，引导您输入新插件的信息。有关插件开发的更多信息，请参阅[此处](/plugins.md)。

```bash
# 项目中已安装 Capacitor CLI
npx cap plugin:generate

# 未安装 Capacitor CLI
npx @capacitor/cli@2.4.7 plugin:generate
```
