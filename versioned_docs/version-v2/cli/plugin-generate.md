# 插件生成

创建新的自定义 Capacitor 插件。该命令将启动一个向导，提示您输入新插件的相关信息。更多关于插件开发的信息请参见[此处](/plugins.md)。

```bash
# 项目中已安装 Capacitor CLI
npx cap plugin:generate

# 未安装 Capacitor CLI
npx @capacitor/cli@2.4.7 plugin:generate
```
