# 构建 Ionic 照片画廊应用

**支持平台**：Web、iOS、Android

Capacitor 能让你轻松构建在 iOS、Android、桌面端和 Web 上原生运行的应用程序。本指南将带你构建一个支持所有平台的完整照片画廊应用。

> 本指南现已在 Ionic 官方文档中维护，[最新版本](https://ionicframework.com/docs/intro/next) 包含 Angular、React 和 Vue 的教程。
