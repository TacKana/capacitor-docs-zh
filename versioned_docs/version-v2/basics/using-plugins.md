# 使用 Capacitor 插件

Capacitor 内置了许多插件，例如 [Camera（相机）](/apis/camera.md) 和 [Filesystem（文件系统）](/apis/filesystem.md)。

但是，您的应用很可能需要访问比 Capacitor 开箱即用提供的更多原生功能。

幸运的是，您可以通过编写少量原生代码，然后按照 [插件指南](/plugins.md) 将其暴露为 Capacitor 插件，从而快速添加自己的原生功能。

## 查找插件

社区已经为各种原生功能构建了许多插件，您可以轻松地将它们添加到您的应用中。

请参阅我们的 [社区插件](/plugins/community.md) 列表以获取一些选择。

## 使用 Cordova 和 Ionic Native 插件

Capacitor 还支持广泛的 [Cordova 插件](/cordova/using-cordova-plugins.md)，因此有很多选择。
