# Capacitor：面向 Web 应用的跨平台原生运行时

Capacitor 是一个跨平台原生运行时，可轻松构建能够在 iOS、Android 和 Web 上原生运行的现代 Web 应用。作为混合应用发展的新阶段，Capacitor 创建的是 **<a href="https://webnative.tech" class="unstyled-link testing">Web Native 应用</a>**，为希望在保持 Web 优先的同时、在需要时仍能完全访问原生 SDK 的团队提供了一种现代的原生容器方案。

![Capacitor 徽标与背景](/img/v3/docs/capacitor-index.png)

## 简介

Capacitor 提供了一套以 Web 为核心的统一 API，使应用能够尽可能贴近 Web 标准，同时在支持的平台上访问丰富的原生设备功能。通过为 iOS 的 Swift、Android 的 Java 以及 Web 的 JavaScript 提供的简单插件 API，添加原生功能变得轻而易举。

Capacitor 是 [Apache Cordova](https://cordova.apache.org/) 和 [Adobe PhoneGap](https://phonegap.com/) 的精神续作，并借鉴了其他流行的跨平台工具如 [React Native](http://facebook.github.io/react-native/) 和 [Turbolinks](https://github.com/turbolinks/turbolinks) 的灵感，但其核心目标完全聚焦于让现代 Web 应用能够轻松运行在所有主流平台上。Capacitor 向后兼容许多现有的 [Cordova 插件](https://cordova.apache.org/plugins/)。

## 快速开始

开始使用 Capacitor 非常简单！Capacitor 可以集成到任何现有的现代 JavaScript 项目中，也可以从头创建一个全新的 Capacitor 项目。

请按照 <a href="/docs/getting-started">安装指南</a> 开始构建您的应用。
