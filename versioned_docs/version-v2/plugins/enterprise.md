# 企业级 Capacitor

Capacitor 为关键任务型企业应用提供强大的原生安全、身份验证和数据管理解决方案及插件。

此外，团队还能获得备受客户喜爱的技术支持，并可选择咨询服务，以确保您的下一个重大应用项目取得成功并持续保持成功，无论移动生态系统如何变化。

[Capacitor 企业版服务](/plugins/enterprise.md) 围绕一套强大的原生 Capacitor 插件展开：[Auth Connect](https://ionicframework.com/enterprise/auth-connect) 用于安全身份验证，[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 用于加密会话和用户身份管理，以及 [Offline Storage](https://ionicframework.com/enterprise/offline-storage) 用于高性能、加密的 SQL 数据存储。

每个解决方案都能与在 Capacitor 中运行的任何现代 Web 技术栈完美配合，且无需 Ionic Framework。

## 企业级身份验证

借助 [Auth Connect](https://ionicframework.com/enterprise/auth-connect) 和 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，您可以使用移动安全最佳实践（如生物识别认证和加密会话存储）连接到任何 OAuth 或 OpenID 提供商（Auth0、Ping、AWS Cognito、Azure Active Directory 等）。

Auth Connect 为众多企业和消费级身份提供商提供了一个简单、可移植的身份验证 API，仅用几行 JavaScript 代码即可实现复杂的原生身份验证流程。

结合 Identity Vault（见下文），Capacitor 应用能够实现最强大、最安全且最*正确*的企业级身份验证流程。

## 安全的身份管理

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 与 Auth Connect 协同工作，保护敏感的用户和身份数据（如身份验证令牌），其背后支持最广泛的 iOS 和 Android 设备的生物识别认证。Identity Vault 超越了典型的指纹或面部扫描插件，可与每个平台上的安全飞地（secure enclaves）和加密服务配合工作。这一点至关重要，因为仅仅扫描指纹或面部身份（就像大多数社区插件所做的那样）无法提供足够的安全性，也无法有效防范越狱或其他攻击。

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 也可独立使用，并与任何现有的身份验证流程集成，以提供高级会话安全。

## 离线和加密数据存储

通过 [Offline Storage](https://ionicframework.com/enterprise/offline-storage)，Capacitor 开发者可以访问一个高性能、加密的 SQL 数据存储系统，用于在 iOS 和 Android 上构建功能强大且安全的支持离线功能的应用。

结合 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，开发者可以安全管理数据加密密钥，并实现设备端数据管理的最佳实践，而无需编写任何原生代码。
