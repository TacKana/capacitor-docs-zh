# Capacitor 社区插件

社区已经构建了许多 Capacitor 插件来为你的应用添加功能。

有多种方式可以找到社区插件：在 Web 或 npm 上搜索已发布的插件，或者浏览官方的 [Capacitor 社区](https://github.com/capacitor-community) GitHub 组织和 [NPM scope](https://npmjs.com/~capacitor-community)，获取经过筛选的优质、社区支持的 Capacitor 插件列表。

## Capacitor 社区

官方的 [Capacitor 社区](https://github.com/capacitor-community) GitHub 组织和 [NPM scope](https://npmjs.com/~capacitor-community) 是一项新的努力，旨在筛选出最佳的 Capacitor 社区支持插件，并提供一致的项目和代码规范。Capacitor 核心团队负责协助工作，但并不正式维护该组织中的任何插件。

<a href="https://github.com/capacitor-community/" class="ui-button">探索 Capacitor 社区</a>

## Cordova 生态系统

Capacitor 支持大多数 Cordova 插件，因此开发者可以在其 Capacitor 应用中使用[数百个现有的 Cordova 插件](https://cordova.apache.org/plugins/)。虽然某些 Cordova 插件与 Capacitor [不兼容](/cordova/known-incompatible-plugins.md)，但大多数都是兼容的，因此如果没有现有的 Capacitor 专属插件，值得尝试使用 Cordova 插件。

另请参阅 [Ionic Native 社区](https://ionicframework.com/docs/native/community)插件，其中列出了带有便捷封装的主要 Cordova 插件，使其更易于在应用中使用。你可以使用 Ionic Native 社区封装，或者将列表作为优质 Cordova 插件的参考。
