# Capacitor 社区插件

社区已经构建了许多 Capacitor 插件，为您的应用增添功能。

有多种方式可以找到社区插件：可以通过网络或 npm 搜索已发布的插件，或者浏览官方的 [Capacitor Community](https://github.com/capacitor-community) GitHub 组织和 [npm 作用域](https://npmjs.com/~capacitor-community)，那里有一份精心筛选的、由社区支持的高质量 Capacitor 插件列表。

## Capacitor Community

官方的 [Capacitor Community](https://github.com/capacitor-community) GitHub 组织和 [npm 作用域](https://npmjs.com/~capacitor-community) 是一项新的举措，旨在精选最佳的 Capacitor 社区支持插件，并提供一致的项目和代码规范。Capacitor 核心团队会协助相关工作，但官方并不维护此组织内的任何插件。

[探索 Capacitor Community &#8250;](https://github.com/capacitor-community)

## Capawesome

[Capawesome](https://github.com/capawesome-team) GitHub 组织是我们的另一位 Ionic 开发专家推出的另一个社区项目，目标是为整个社区提供出色的 Capacitor 插件。其中包括 [capacitor-firebase](https://github.com/capawesome-team/capacitor-firebase) 和 [capacitor-mlkit](https://github.com/capawesome-team/capacitor-mlkit)。您可以查看 [Capawesome 博客](https://capawesome.io/blog/) 或关注其 [Twitter](https://twitter.com/capawesomeio) 以获取最新信息。

## Cordova 生态系统

Capacitor 支持大多数 Cordova 插件，因此开发者可以在他们的 Capacitor 应用中使用[数百个现有的 Cordova 插件](https://cordova.apache.org/plugins/)。尽管某些 Cordova 插件与 Capacitor [不兼容](/plugins/cordova.md#known-incompatible-plugins)，但大多数是兼容的，所以如果没有现成的 Capacitor 专用插件，值得尝试一下 Cordova 插件。

另请查看 [`@awesome-cordova-plugins` 社区](https://github.com/danielsogl/awesome-cordova-plugins) 插件，该列表主要包含 Cordova 插件，并提供了便捷的封装器，使它们在您的应用中更易于使用。您可以随意使用 `@awesome-cordova-plugins` 社区的封装器，或者仅将该列表作为高质量 Cordova 插件的参考。
