# Capacitor：适用于 Web 应用的跨平台原生运行时

Capacitor 是一个跨平台原生运行时，它让使用现代 Web 工具构建高性能移动应用变得简单，这些应用能在 iOS、Android 等平台上原生运行。作为混合应用的下一次演进，Capacitor 创造了 **Web 原生应用**，为那些希望以 Web 优先方式构建应用，同时在需要时又不牺牲对原生 SDK 完全访问权的团队，提供了一种现代的原生容器方案。

![Capacitor 徽标与背景](/img/v6/docs/capacitor-index.png)

## 简介

Capacitor 提供了一套一致、以 Web 为中心的 API，让应用尽可能贴近 Web 标准，同时在支持的平台上访问丰富的原生设备功能。如果某个功能在浏览器中有效，那么在使用 Capacitor 的移动应用中很可能也同样有效。通过为 iOS 的 Swift、Android 的 Java 以及 Web 的 JavaScript 提供插件 API，添加原生功能变得直接明了。

## 快速开始

开始使用 Capacitor 非常容易！Capacitor 可以集成到任何现有的现代 JavaScript 项目中，也可以从头创建一个全新的 Capacitor 项目。

请按照 [安装指南](/main/getting-started/installation.md) 开始构建您的应用。
