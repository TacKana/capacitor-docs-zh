# Capacitor：跨平台原生运行时，助力 Web 应用

Capacitor 是一个跨平台原生运行时，它使得使用现代 Web 工具构建在 iOS、Android 等平台上原生运行的高性能移动应用变得简单。作为混合应用的下一次演进，Capacitor 创建了 **Web 原生应用**，为希望以 Web 为先、同时在需要时仍能完全访问原生 SDK 的团队提供了一种现代原生容器方案。

![Capacitor 标志和背景图](/img/v6/docs/capacitor-index.png)

## 介绍

Capacitor 提供了一套一致的、以 Web 为中心的 API，使应用能够尽可能地遵循 Web 标准，同时在支持它们的平台上访问丰富的原生设备功能。如果它在浏览器中能工作，那么在使用 Capacitor 时，它在移动应用中也大概率能工作。通过适用于 iOS 的 Swift、适用于 Android 的 Java 以及适用于 Web 的 JavaScript 的插件 API，添加原生功能变得简单直接。

## 开始使用

开始使用 Capacitor 非常简单！Capacitor 可以集成到任何现有的现代 JavaScript 项目中，也可以从头开始创建一个全新的 Capacitor 项目。

请遵循[安装指南](/main/getting-started/installation.md)开始构建您的应用。
