# Capacitor CLI - cap add

向你的应用添加原生平台项目。

```bash
npx cap add <平台>
```

<strong>输入参数：</strong>

- `platform`（必需）：`android`, `ios`

<strong>选项：</strong>

仅 iOS：
- `--packagemanager`: `SPM`, `Cocoapods`
