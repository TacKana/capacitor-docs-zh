按照以下步骤为 Angular、React 或 Vue 创建一个**全新初始项目**。

<video style={{borderRadius:5}} width="100%" controls>
      <source src={require('@site/static/video/new-project.mp4').default} type="video/mp4" />
</video>

1. 为项目创建文件夹并在 VS Code 中打开
2. 点击侧边栏的 `Ionic` 图标
3. 选择你偏好的前端框架
4. 从多种初始模板中选择
5. 输入项目`名称`并按`Enter`键

经过短暂的安装过程后，你的初始项目就准备就绪了。

你也可以为[现有网页应用添加原生支持](web-to-native)，不限使用何种框架。
