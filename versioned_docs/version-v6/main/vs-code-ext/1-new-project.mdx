按照以下步骤为 Angular、React 或 Vue 创建一个**新的启动项目**。

<video style={{borderRadius:5}} width="100%" controls>
      <source src={require('@site/static/video/new-project.mp4').default} type="video/mp4" />
</video>

1. 为您的项目创建一个文件夹，并在 VS Code 中打开
2. 点击侧边菜单中的 `Ionic` 徽标
3. 选择您偏好的框架
4. 从各种启动项目中选择
5. 为您的项目输入 `name` 并按下 `Enter`

短暂的安装过程后，您的启动项目就可以开始工作了。

您也可以[将现有 Web 应用原生化](web-to-native)，无论使用什么框架。
