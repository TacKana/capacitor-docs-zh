按照以下步骤，为 **Angular、React 或 Vue** 创建一个**新的入门项目**。

<video style={{borderRadius:5}} width="100%" controls>
      <source src={require('@site/static/video/new-project.mp4').default} type="video/mp4" />
</video>

1. 为您的项目创建一个文件夹，并在 VS Code 中打开它
2. 点击侧边栏菜单中的 `Ionic` 图标
3. 选择您偏爱的框架
4. 从多种入门项目模板中选择一个
5. 输入项目的 `name` 并按 `Enter` 键

经过简短的安装过程后，您的入门项目就准备就绪，可以开始工作了。

无论您使用何种框架，您也可以[将您现有的 Web 应用转换为原生应用](web-to-native)。
