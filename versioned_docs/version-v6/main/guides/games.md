# 使用 Capacitor 进行游戏开发

Capacitor 是构建跨平台游戏的优秀平台。通过对 WebGL 和 canvas 渲染的广泛支持，开发者可以在现代移动设备上构建高性能的游戏体验。

我们通常建议使用游戏引擎进行构建，它将为常见的游戏对象和功能提供原语，不过开发者也可以从头开始构建。

## Phaser + Capacitor

[Phaser](https://phaser.io/) 是最流行的基于 Web 的游戏开发引擎之一。

要开始使用 Capacitor 和 Phaser 构建游戏，请参阅[将您的 Phaser 游戏通过 Capacitor 带到 iOS 和 Android](https://phaser.io/tutorials/bring-your-phaser-game-to-ios-and-android-with-capacitor)
