# 构建 Ionic 相册应用

**平台支持**：网页、iOS、Android

Capacitor 能轻松构建可在 iOS、Android、桌面端和网页端原生运行的 Web 应用。本指南将带您构建一个全平台兼容的完整相册应用。

> 本指南最新维护版本已迁移至 Ionic 官方文档站，[最新版](https://ionicframework.com/docs/intro/next)包含 Angular、React 和 Vue 版本的教程。
