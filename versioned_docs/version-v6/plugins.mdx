# Capacitor插件

Capacitor中的插件能让JavaScript直接调用原生API接口。

<br />

通过插件机制，Web应用能够获得原生API的全部能力。插件封装了各平台原生API的差异性，为JavaScript提供统一的跨平台接口。

Capacitor的插件架构还使得原生开发者和Web开发者能够协作开发应用的不同模块。

Capacitor会自动在客户端生成JavaScript调用层，因此大多数插件只需为iOS平台编写Swift/Obj-C代码，为Android平台编写Java/Kotlin代码即可。当然，开发者也可以为插件添加自定义的JavaScript逻辑。
