# Capacitor 插件

Capacitor 中的插件使得 JavaScript 能够直接与原生 API 交互。

<br />

通过插件，Web 应用可以充分利用原生 API 的强大功能。插件封装了跨平台可能使用不同 API 的原生操作，并向 JavaScript 暴露出一致、统一的跨平台 API。

此外，Capacitor 的插件功能让拥有传统原生开发者和 Web 开发者混合的团队能够协同工作，各自负责应用的不同部分。

Capacitor 会自动在客户端生成 JavaScript 钩子，因此大多数插件只需要为 iOS 使用 Swift/Obj-C 和/或为 Android 使用 Java/Kotlin 即可。当然，为插件添加自定义 JavaScript 代码也是可以的。
