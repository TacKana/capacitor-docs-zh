# 构建 Ionic 照片库应用

**支持平台**：Web、iOS、Android

Capacitor 可以轻松构建能在 iOS、Android、桌面端和 Web 上原生运行的 Web 应用。在本指南中，我们将构建一个可在所有平台上运行的完整照片库应用。

> 本指南现已在 Ionic 文档中进行维护。[最新版本](https://ionicframework.com/docs/intro/next) 包含了针对 Angular、React 和 Vue 的教程。
