# Capacitor 应用中的广告

许多消费类应用需要显示广告以产生收入。

移动应用最流行的广告网络是 Google AdMob，并且有一个[社区维护的 Capacitor 插件](https://github.com/capacitor-community/admob)可以直接使用：

```shell
npm install @capacitor-community/admob
npx cap update
```

然后，按照 [Capacitor Community AdMob](https://github.com/capacitor-community/admob) 的文档进行配置和使用。

这篇[社区文章](https://medium.com/enappd/implement-admob-in-ionic-react-capacitor-apps-ebc7af360b41)也是该插件实际应用的有用参考（尽管它使用的是旧版本插件）。
