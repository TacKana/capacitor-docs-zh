# PWA 元素

部分 Capacitor 插件（例如 `Camera` 或 `Toast`）在非原生环境下运行时提供了基于 Web 的 UI。例如，在 Web 上调用 `Camera.getPhoto()` 会加载一个响应式的拍照体验：

![PWA 元素](/img/v6/docs/pwa-elements.png)

此 UI 是使用 Web 组件实现的。由于这些元素被 [Shadow DOM](https://developer.mozilla.org/en-US/docs/Web/Web_Components/Using_shadow_DOM) 封装，它们不会与你自己的 UI 产生冲突。

## 安装

要启用这些控件，必须在应用中添加 `@ionic/pwa-elements`。

典型安装方式包括导入包并注册元素，或在应用的 `index.html` 的 `<head>` 中添加脚本标签：

#### 导入 PWA 元素

```bash
npm install @ionic/pwa-elements
```

然后，根据你选择的框架，在适当时机导入元素加载器并调用：

##### React

`main.tsx`、`index.tsx` 或 `index.js`：

```tsx
import { defineCustomElements } from '@ionic/pwa-elements/loader';

// 在渲染调用前调用元素加载器
defineCustomElements(window);
```

##### Vue

`main.ts`：

```typescript
// 在 createApp() 行之前
import { defineCustomElements } from '@ionic/pwa-elements/loader';
defineCustomElements(window);
```

##### Angular

`main.ts`：

```typescript
import { defineCustomElements } from '@ionic/pwa-elements/loader';
// 在 bootstrapModule/bootstrapApplication 调用前调用元素加载器
defineCustomElements(window);
if (environment.production) {
  enableProdMode();
}
```

#### 通过脚本标签引入

PWA 元素可以通过 `index.html` 中的脚本标签引入。但请注意，这种方式在离线场景下无法工作：

```html
<script
  type="module"
  src="https://unpkg.com/@ionic/pwa-elements@latest/dist/ionicpwaelements/ionicpwaelements.esm.js"
></script>
<script
  nomodule
  src="https://unpkg.com/@ionic/pwa-elements@latest/dist/ionicpwaelements/ionicpwaelements.js"
></script>
```
