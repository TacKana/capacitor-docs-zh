# Capacitor CLI - cap open

在指定的原生 IDE 中打开项目工作区（iOS 使用 Xcode，Android 使用 Android Studio）。

```bash
npx cap open <platform>
```

<strong>参数:</strong>

- `platform` (必填): `android`, `ios`
