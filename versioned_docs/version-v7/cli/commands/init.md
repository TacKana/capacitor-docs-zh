# Capacitor CLI - cap init

通过提供应用名称、应用 ID 以及可选的现有 Web 应用目录来初始化 Capacitor 配置。

```bash
npx cap init <appName> <appID>
```

<strong>输入参数：</strong>

- `appName`（必需）：应用的名称
- `appID`（必需）：应用的应用 ID，例如 `com.example.appname`

<strong>选项：</strong>

- `--web-dir <value>`：要用于初始化的现有 Web 应用程序目录
