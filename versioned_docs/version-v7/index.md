# Capacitor：跨平台原生运行时，用于 Web 应用

Capacitor 是一个跨平台原生运行时，它使得使用现代 Web 工具构建在 iOS、Android 等平台上原生运行的高性能移动应用变得简单。代表混合应用的下一次演进，Capacitor 创建了 **Web 原生应用**，为那些希望以 Web 为先、同时在需要时也不牺牲对原生 SDK 完全访问权限的团队提供了一种现代原生容器方法。

![Capacitor 标志和背景](/img/v6/docs/capacitor-index.png)

## 简介

Capacitor 提供了一组一致的、以 Web 为中心的 API，使应用能够尽可能贴近 Web 标准，同时在支持它们的平台上访问丰富的原生设备功能。如果它在浏览器中能工作，那么在使用 Capacitor 的移动应用中很可能也能工作。通过适用于 iOS 的 Swift、Android 的 Java 和 Web 的 JavaScript 的插件 API，添加原生功能非常简单。

## 开始使用

开始使用 Capacitor 非常简单！Capacitor 可以嵌入到任何现有的现代 JavaScript 项目中，也可以从头开始创建一个全新的 Capacitor 项目。

按照[安装指南](/main/getting-started/installation.md)开始构建您的应用。
