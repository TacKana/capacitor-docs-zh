# Capacitor：适用于 Web 应用的多平台原生运行时

Capacitor 是一个跨平台原生运行时，可让你轻松构建高性能的移动应用，这些应用能够在 iOS、Android 等平台上原生运行，同时使用现代化的 Web 工具链。作为混合应用的新一代演进，Capacitor 创建了 **Web 原生应用**，为希望在 Web 优先开发方式下不牺牲原生 SDK 完整访问能力的团队提供了现代化的原生容器方案。

![Capacitor 标志与背景](/img/v6/docs/capacitor-index.png)

## 简介

Capacitor 提供了一套统一且以 Web 为核心的 API，使应用能够尽可能贴近 Web 标准，同时在支持的平台上访问丰富的原生设备功能。如果某个功能在浏览器中能够运行，那么在使用 Capacitor 的移动应用中很可能也能运行。通过为 iOS 的 Swift、Android 的 Java 以及 Web 的 JavaScript 提供插件 API，添加原生功能变得非常简单。

## 快速开始

开始使用 Capacitor 很容易！Capacitor 可以无缝集成到任何现有的现代 JavaScript 项目中，也可以从头创建一个全新的 Capacitor 项目。

请按照 [安装指南](/main/getting-started/installation.md) 开始构建你的应用。
