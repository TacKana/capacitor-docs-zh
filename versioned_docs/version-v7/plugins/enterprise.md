# Capacitor 企业版

Capacitor 为关键任务型企业应用提供强大的原生安全、认证和数据管理解决方案及插件。

此外，团队还可获得备受客户好评的技术支持以及可选的专业咨询服务，确保您的下一个重要应用项目取得成功并长期保持稳定，无论移动生态系统如何变化。

[Capacitor 企业版服务](/plugins/enterprise.md) 围绕一系列强大的原生 Capacitor 插件展开：[Auth Connect](https://ionicframework.com/enterprise/auth-connect) 用于安全认证，[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 用于加密会话和用户身份管理，[Offline Storage](https://ionicframework.com/enterprise/offline-storage) 则提供高性能的加密 SQL 数据存储。

每个解决方案都能与在 Capacitor 中运行的任何现代 Web 技术栈完美配合，且无需依赖 Ionic Framework。

## 企业级身份认证

通过 [Auth Connect](https://ionicframework.com/enterprise/auth-connect) 和 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，您可以使用移动安全最佳实践（如生物识别认证和加密会话存储）连接到任何 OAuth 或 OpenID 提供商（Auth0、Ping、AWS Cognito、Azure Active Directory 等）。

Auth Connect 为企业级和消费者身份提供商提供简单、可移植的身份认证 API，仅需几行 JavaScript 代码即可实现复杂的原生认证流程。

结合 Identity Vault（见下文），Capacitor 应用能够实现最强大、最安全且最 _规范_ 的企业级认证流程。

## 安全的身份管理

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 与 Auth Connect 协同工作，保护敏感的用户和身份数据（如认证令牌），其依托于生物识别认证，支持最广泛的 iOS 和 Android 设备。Identity Vault 超越了典型的指纹或面部扫描插件，能够与各平台的安全飞地和加密服务协同工作。这一点至关重要，因为仅扫描指纹或面部身份（如大多数社区插件所做的那样）无法提供足够的安全性，也无法有效防范越狱或其他恶意攻击。

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 也可独立使用，并与任何现有认证流程集成，以提供高级的会话安全保障。

## 离线和加密数据存储

借助 [Offline Storage](https://ionicframework.com/enterprise/offline-storage)，Capacitor 开发者可以使用高性能的加密 SQL 数据存储系统，在 iOS 和 Android 上构建强大且安全的离线应用。

结合 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，开发者可以安全地管理数据加密密钥，并实施设备端数据管理的最佳实践，而无需编写一行原生代码。
