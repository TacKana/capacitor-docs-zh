# Capacitor 社区插件

社区已经构建了许多 Capacitor 插件来为你的应用增加功能。

有几种方法可以找到社区插件：搜索 Web 或 npm 以查找任何已发布的插件，或者浏览官方的 [Capacitor Community](https://github.com/capacitor-community) GitHub 组织和 [npm scope](https://npmjs.com/~capacitor-community)，以获取经过精选的高质量、社区支持的 Capacitor 插件列表。

## Capacitor Community

官方的 [Capacitor Community](https://github.com/capacitor-community) GitHub 组织和 [npm scope](https://npmjs.com/~capacitor-community) 是一项新的努力，旨在精选最佳的 Capacitor 社区支持插件，并提供一致的项目和代码规范。Capacitor 核心团队促进工作，但不正式维护该组织中的任何插件。

[探索 Capacitor Community &#8250;](https://github.com/capacitor-community)

## Capawesome

[Capawesome](https://github.com/capawesome-team) GitHub 组织是另一个由我们的 Ionic 开发者专家之一发起的社区项目，旨在为整个社区提供出色的 Capacitor 插件。
其中包括 [capacitor-firebase](https://github.com/capawesome-team/capacitor-firebase) 和 [capacitor-mlkit](https://github.com/capawesome-team/capacitor-mlkit) 等。
查看 [Capawesome Blog](https://capawesome.io/blog/) 或 [Twitter](https://twitter.com/capawesomeio) 以获取最新信息。

## Cordova 生态系统

Capacitor 支持大多数 Cordova 插件，因此开发者可以在他们的 Capacitor 应用中使用[数百个现有的 Cordova 插件](https://cordova.apache.org/plugins/)。虽然某些 Cordova 插件与 Capacitor [不兼容](/plugins/cordova.md#已知不兼容的插件)，但大多数是可用的，因此如果还没有现有的 Capacitor 特定插件，值得尝试。

另请参阅 [`@awesome-cordova-plugins` Community](https://github.com/danielsogl/awesome-cordova-plugins) 插件，获取主要 Cordova 插件的列表，这些插件提供了方便的包装器，使其更易于在应用中使用。你可以自由使用 `@awesome-cordova-plugins` Community 包装器，或简单地将该列表作为优质 Cordova 插件的参考。
