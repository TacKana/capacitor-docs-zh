# Capacitor：跨平台原生运行时，适用于 Web 应用

Capacitor 是一个跨平台原生运行时，使得使用现代 Web 工具构建在 iOS、Android 等平台上原生运行的高性能移动应用变得简单。作为混合应用的下一代演进，Capacitor 创建了 **Web 原生应用**，为希望以 Web 为先、同时在需要时又不放弃对原生 SDK 完全访问权限的团队提供了现代化的原生容器方案。

![Capacitor Logo 与背景](/img/v5/docs/capacitor-index.png)

## 简介

Capacitor 提供了一套一致的、以 Web 为中心的 API，使应用能够尽可能贴近 Web 标准，同时能够在支持这些功能的平台上访问丰富的原生设备特性。如果它在浏览器中能工作，那么在使用 Capacitor 时，它很可能也能在移动应用中工作。通过适用于 iOS 的 Swift、适用于 Android 的 Java 以及适用于 Web 的 JavaScript 的插件 API，添加原生功能非常直接。

## 开始使用

开始使用 Capacitor 非常简单！Capacitor 可以加入到任何现有的现代 JavaScript 项目中，也可以从头创建一个全新的 Capacitor 项目。

请按照[安装指南](/main/getting-started/installation.md)开始构建您的应用。
