# Capacitor：面向 Web 应用的多平台原生运行时

Capacitor 是一款跨平台原生运行时，让您能够轻松构建高性能的移动应用，这些应用能够使用现代 Web 工具链在 iOS、Android 等平台上原生运行。作为混合应用的新一代演进方向，Capacitor 创造了 **Web Native 应用**，为那些希望以 Web 优先的方式构建应用，同时又在需要时能够完全访问原生 SDK 的团队提供了现代化的原生容器方案。

![Capacitor 标志与背景](/img/v5/docs/capacitor-index.png)

## 简介

Capacitor 提供了一套专注于 Web 的一致性 API，让应用能够尽可能贴近 Web 标准，同时在各支持平台上访问丰富的原生设备功能。如果一个功能在浏览器中能够运行，那么在使用 Capacitor 时，它很可能也能在移动应用中运行。通过为 iOS 的 Swift、Android 的 Java 以及 Web 的 JavaScript 提供的插件 API，添加原生功能变得简单直接。

## 开始使用

开始使用 Capacitor 十分简单！Capacitor 可以轻松集成到任何现有的现代化 JavaScript 项目中，也可以从头创建一个全新的 Capacitor 项目。

请遵循 [安装指南](/main/getting-started/installation.md) 开始构建您的应用。
