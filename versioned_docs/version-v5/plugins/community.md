# Capacitor 社区插件

社区已经构建了许多 Capacitor 插件来为您的应用添加功能。

有多种方法可以找到社区插件：搜索 Web 或 npm 来查找任何已发布的插件，或者探索官方的 [Capacitor Community](https://github.com/capacitor-community) GitHub 组织和 [npm scope](https://npmjs.com/~capacitor-community) 以获取经过精选的高质量、社区支持的 Capacitor 插件列表。

## Capacitor 社区

官方的 [Capacitor Community](https://github.com/capacitor-community) GitHub 组织和 [npm scope](https://npmjs.com/~capacitor-community) 是一项新的努力，旨在精选最佳的社区支持的 Capacitor 插件，并提供一致的项目和代码约定。Capacitor 核心团队促进工作，但不在该组织中正式维护任何插件。

[探索 Capacitor 社区 &#8250;](https://github.com/capacitor-community)

## Capawesome

[Capawesome](https://github.com/capawesome-team) GitHub 组织是另一个由我们的 Ionic 开发者专家之一发起的社区项目，旨在为整个社区提供出色的 Capacitor 插件。
其中包括 [capacitor-firebase](https://github.com/capawesome-team/capacitor-firebase) 和 [capacitor-mlkit](https://github.com/capawesome-team/capacitor-mlkit) 等。
查看 [Capawesome 博客](https://capawesome.io/blog/) 或 [Twitter](https://twitter.com/capawesomeio) 以获取最新信息。

## Cordova 生态系统

Capacitor 支持大多数 Cordova 插件，因此开发者可以在其 Capacitor 应用中使用[数百个现有的 Cordova 插件](https://cordova.apache.org/plugins/)。虽然某些 Cordova 插件[不兼容](/plugins/cordova.md#known-incompatible-plugins) Capacitor，但大多数是兼容的，因此如果没有现成的 Capacitor 专用插件，值得尝试一下。

另请参见 [`@awesome-cordova-plugins` 社区](https://github.com/danielsogl/awesome-cordova-plugins) 插件，该列表主要包含带有便捷包装器以便于在应用中使用的 Cordova 插件。请随意使用 `@awesome-cordova-plugins` 社区包装器，或仅将该列表用作高质量 Cordova 插件的参考。
