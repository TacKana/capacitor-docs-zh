# Capacitor 企业版

Capacitor 为关键任务型企业应用提供了强大的原生安全、身份验证和数据管理解决方案及插件。

此外，团队还可以获得深受客户喜爱的技术支持以及可选咨询服务，确保您的下一个重大应用项目取得成功并持续稳定运行，无需顾虑移动生态系统的变化。

[Capacitor 企业版产品](/plugins/enterprise.md)的核心是一套功能强大的原生 Capacitor 插件：[Auth Connect](https://ionicframework.com/enterprise/auth-connect) 用于安全身份验证，[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 用于加密会话和用户身份管理，[Offline Storage](https://ionicframework.com/enterprise/offline-storage) 用于高性能加密 SQL 数据存储。

每个解决方案都能与在 Capacitor 中运行的任何现代 Web 技术栈完美配合，并且不需要 Ionic 框架。

## 企业级身份验证

借助 [Auth Connect](https://ionicframework.com/enterprise/auth-connect) 和 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，您可以使用移动安全最佳实践（如生物特征认证和加密会话存储）连接到任何 OAuth 或 OpenID 提供商（Auth0、Ping、AWS Cognito、Azure Active Directory 等）。

Auth Connect 为众多企业和消费者身份提供商提供了一套简单、可移植的身份验证 API，仅需几行 JavaScript 代码即可实现复杂的原生身份验证流程。

结合 Identity Vault（见下文），Capacitor 应用能够实现最强大、最安全且最 *正确* 的企业级身份验证流程。

## 安全的身份管理

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 与 Auth Connect 协同工作，在支持最广泛 iOS 和 Android 设备的生物特征认证基础上，保护敏感的用户和身份数据（例如身份验证令牌）。Identity Vault 超越了典型的指纹或面部扫描插件，它能与各平台的安全飞地和加密服务协同工作。这一点至关重要，因为仅仅像大多数社区插件那样扫描指纹或面部身份，并不能提供足够的安全性，也无法有效防御越狱或其他攻击。

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 也可以独立使用，并与任何现有的身份验证流程集成，以提供高级的会话安全保障。

## 离线与加密数据存储

通过 [Offline Storage](https://ionicframework.com/enterprise/offline-storage)，Capacitor 开发者可以访问一个高性能、加密的 SQL 数据存储系统，用于在 iOS 和 Android 上构建强大且安全的离线应用。

结合 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，开发者可以安全地管理数据加密密钥，并实施设备端数据管理最佳实践，而无需编写任何原生代码。
