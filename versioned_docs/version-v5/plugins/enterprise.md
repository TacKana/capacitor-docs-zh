# Capacitor 企业版

Capacitor 为关键任务的企业应用提供强大的原生安全、身份验证和数据管理解决方案及插件。

此外，团队还可以获得客户喜爱的支持和可选的咨询服务，以确保您的下一个重大应用项目成功并保持这种状态，无论移动生态系统如何变化。

[Capacitor 的企业版产品](/plugins/enterprise.md) 围绕一套强大的原生 Capacitor 插件展开：[Auth Connect](https://ionicframework.com/enterprise/auth-connect) 用于安全身份验证，[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 用于加密会话和用户身份管理，以及 [Offline Storage](https://ionicframework.com/enterprise/offline-storage) 用于高性能加密 SQL 数据存储。

每个解决方案都能与在 Capacitor 中运行的任何现代 Web 技术栈完美配合，并且不需要 Ionic Framework。

## 企业身份验证

通过 [Auth Connect](https://ionicframework.com/enterprise/auth-connect) 和 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，使用移动安全最佳实践（如生物识别身份验证和加密会话存储）连接到任何 OAuth 或 OpenID 提供商（Auth0、Ping、AWS Cognito、Azure Active Directory 等）。

Auth Connect 为与众多企业和消费级身份提供商的身份验证提供了一个简单、可移植的 API，只需几行 JavaScript 代码即可实现复杂的原生身份验证流程。

与 Identity Vault（见下文）结合，Capacitor 应用可以实现最强大、最安全且_正确_的企业身份验证流程。

## 安全身份管理

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 与 Auth Connect 协同工作，保护敏感的用户和身份数据，如身份验证令牌，并由生物识别身份验证提供支持，支持最广泛的 iOS 和 Android 设备。Identity Vault 超越了典型的指纹或面部扫描插件，与每个平台上的安全 enclave 和加密服务一起工作。这是必要的，因为像大多数社区插件那样简单地扫描指纹或面部身份并不能提供足够的安全性，也无法防止越狱或其他漏洞利用。

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 也可以独立使用，并与任何现有的身份验证流程集成，以提供高级会话安全性。

## 离线和加密数据存储

通过 [Offline Storage](https://ionicframework.com/enterprise/offline-storage)，Capacitor 开发者可以访问一个高性能、加密的 SQL 数据存储系统，用于在 iOS 和 Android 上构建强大且安全的离线应用。

与 [Identity Vault](https://ionicframework.com/enterprise/identity-vault) 结合，开发者可以安全地管理数据加密密钥，并实现设备上数据管理最佳实践，无需编写一行原生代码。
