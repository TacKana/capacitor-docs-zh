# Capacitor CLI

Capacitor 命令行接口（CLI）工具用于开发 Capacitor 应用程序。

## CLI 参考

```bash
npx cap [-V] [-h] [<command>]
```

<strong>选项参数：</strong>

- `-V, --version`（可选）：输出版本号
- `-h, --help`（可选）：输出帮助信息。也可用于单独命令。
