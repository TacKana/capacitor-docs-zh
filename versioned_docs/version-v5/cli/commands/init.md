# Capacitor CLI - cap init

通过提供应用名称、应用 ID 和现有 Web 应用的可选 Web 目录来初始化 Capacitor 配置。

```bash
npx cap init <appName> <appID>
```

<strong>输入：</strong>

- `appName`（必需）：应用的名称
- `appID`（必需）：应用的应用 ID；类似 `com.example.appname`

<strong>选项：</strong>

- `--web-dir <value>`：要与初始化一起使用的现有 Web 应用
