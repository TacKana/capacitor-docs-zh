# 使用 Capacitor 进行游戏开发

Capacitor 是构建跨平台游戏的绝佳平台。凭借对 WebGL 和 canvas 渲染的广泛支持，开发者可以在现代移动设备上构建高性能的游戏体验。

我们通常建议使用游戏引擎进行构建，这将为常见的游戏对象和功能提供基础组件，当然开发者也可以选择从头开始构建。

## Phaser + Capacitor

[Phaser](https://phaser.io/) 是最受欢迎的基于网页的游戏开发引擎之一。

要开始使用 Capacitor 和 Phaser 构建游戏，请参阅 [使用 Capacitor 将你的 Phaser 游戏带到 iOS 和 Android](https://phaser.io/tutorials/bring-your-phaser-game-to-ios-and-android-with-capacitor)
