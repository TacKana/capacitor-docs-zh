您的原生项目的版本、Bundle ID 和显示名称可以轻松读取和修改。

![原生设置](/img/native-settings.png)

点击 `配置` > `属性` 以更改：

- **显示名称** - 主屏幕图标下方显示的名称。
- **Bundle 标识符** - 应用程序的唯一标识符。
- **版本号** - 主要和次要版本号（例如 `2.5`）。
- **构建号** - 通常与版本号关联的构建编号（例如 `3`）。

:::note
更改这些属性之一时，`iOS` 和 `Android` 原生项目都会同步更改。
:::
