按照以下步骤为 Angular、React 或 Vue 创建一个**新的入门项目**。

<video style={{borderRadius:5}} width="100%" controls>
      <source src={require('@site/static/video/new-project.mp4').default} type="video/mp4" />
</video>

1.  为您的项目创建一个文件夹，并在 VS Code 中打开它
2.  点击侧边栏上的 `Ionic` 徽标
3.  选择您偏好的框架
4.  从各种入门项目中进行选择
5.  输入项目的 `名称` 并按 `Enter` 键

经过短暂的安装过程后，您的入门项目即可开始工作。

无论使用何种框架，您也可以[将现有的 Web 应用转换为原生应用](web-to-native)。
