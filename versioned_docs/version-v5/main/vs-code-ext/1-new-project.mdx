按照以下步骤创建适用于 **Angular、React 或 Vue 的新启动项目**。

<video style={{borderRadius:5}} width="100%" controls>
      <source src={require('@site/static/video/new-project.mp4').default} type="video/mp4" />
</video>

1. 为项目创建文件夹并在 VS Code 中打开
2. 点击侧边栏的 `Ionic` 图标
3. 选择您偏好的前端框架
4. 从多种启动模板中选择
5. 输入项目 `name` 并按 `Enter` 键

经过简短的安装过程后，您的启动项目即可开始开发。

您还可以为[现有网页应用添加原生支持](web-to-native)，无论使用何种框架都能实现。
