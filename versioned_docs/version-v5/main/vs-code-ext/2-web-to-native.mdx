你可以使用 Capacitor 将**任何** Web 应用或网站转换为原生应用（通常只需几分钟即可完成）。

<video style={{borderRadius:5}} width="100%" controls>
      <source src={require('@site/static/video/existing-project.mp4').default} type="video/mp4" />
</video>

1. 点击侧边栏的 `Ionic` 图标
2. 选择 `Add Capacitor Integration`
3. 选择 `Add iOS Project` 添加原生 **XCode** 项目
4. 选择 `Add Android Project` 添加原生 **Android Studio** 项目
