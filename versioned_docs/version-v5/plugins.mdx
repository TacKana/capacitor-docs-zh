# Capacitor 插件

Capacitor 中的插件使 JavaScript 能够直接与原生 API 进行交互。

<br />

通过插件，Web 应用可以充分利用原生 API 的强大功能。插件封装了常见的原生操作，这些操作在不同平台上可能使用完全不同的 API，同时向 JavaScript 暴露出一致且跨平台的 API。

此外，Capacitor 的插件功能使得传统原生开发者和 Web 开发者可以协同工作，各自负责应用的不同部分。

Capacitor 会自动在客户端生成 JavaScript 钩子，因此大多数插件只需要为 iOS 使用 Swift/Obj-C 和/或为 Android 使用 Java/Kotlin。当然，为插件添加自定义 JavaScript 也是可行的。
