# Capacitor 插件

Capacitor 中的插件使 JavaScript 能够直接与原生 API 进行交互。

<br />

通过插件，Web 应用可以充分利用原生 API 的强大功能。插件封装了跨平台可能使用截然不同 API 的常见原生操作，同时向 JavaScript 暴露一致、跨平台的 API。

此外，Capacitor 的插件功能使得由传统原生开发者和 Web 开发者组成的团队能够共同协作开发应用的不同部分。

Capacitor 会自动在客户端生成 JavaScript 钩子，因此大多数插件仅需要使用 Swift/Obj-C 处理 iOS 和/或 Java/Kotlin 处理 Android。当然，为插件添加自定义 JavaScript 也是可行的。
