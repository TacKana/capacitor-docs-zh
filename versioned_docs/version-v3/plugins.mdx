# Capacitor 插件

Capacitor 中的插件让 JavaScript 能够直接与原生 API 进行交互。

<br />

通过插件，Web 应用能够充分利用原生 API 的全部能力。插件封装了那些在不同平台上 API 可能差异很大的常见原生操作，同时向 JavaScript 暴露出一致、跨平台的 API。

此外，Capacitor 的插件功能使得同时拥有传统原生开发者和 Web 开发者的团队能够协同工作，各自负责应用的不同部分。

Capacitor 会自动在客户端生成 JavaScript 钩子，因此大多数插件只需为 iOS 使用 Swift/Obj-C 和/或为 Android 使用 Java/Kotlin。当然，为插件添加自定义 JavaScript 也是可行的。
