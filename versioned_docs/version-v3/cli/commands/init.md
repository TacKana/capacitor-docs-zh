# Capacitor CLI - cap init

通过提供应用名称、应用 ID 以及可选的现有 Web 应用 Web 目录，初始化 Capacitor 配置。

```bash
npx cap init <appName> <appID>
```

<strong>输入参数：</strong>

- `appName`（必填）：应用的名称
- `appID`（必填）：应用的 App ID，例如 `com.example.appname`

<strong>选项：</strong>

- `--web-dir <value>`：要与初始化一起使用的现有 Web 应用
