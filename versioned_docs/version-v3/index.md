# Capacitor：适用于 Web 应用的跨平台原生运行时

Capacitor 是一个跨平台原生运行时，可以轻松构建在 iOS、Android 和 Web 上原生运行的现代 Web 应用。作为混合应用演进的新阶段，Capacitor 创造了 **Web Native 应用**，为那些希望以 Web 优先方式构建应用，同时又不放弃在需要时完整访问原生 SDK 能力的团队，提供了一种现代的原生容器方案。

![Capacitor 标志和背景图](/img/v3/docs/capacitor-index.png)

## 简介

Capacitor 提供了一套以 Web 为核心的统一 API，让应用能够尽可能遵循 Web 标准，同时在使用支持的平台时充分利用丰富的原生设备功能。通过简单的插件 API，可以轻松为 iOS 上的 Swift、Android 上的 Java 以及面向 Web 的 JavaScript 添加原生功能。

Capacitor 在精神上延续了 [Apache Cordova](https://cordova.apache.org/) 和 [Adobe PhoneGap](https://phonegap.com/) 的理念，并从 [React Native](http://facebook.github.io/react-native/) 和 [Turbolinks](https://github.com/turbolinks/turbolinks) 等其他流行的跨平台工具中汲取灵感，但其核心目标完全专注于让现代 Web 应用能够轻松运行在所有主流平台上。Capacitor 向后兼容许多现有的 [Cordova 插件](https://cordova.apache.org/plugins/)。

## 开始使用

开始使用 Capacitor 非常简单！Capacitor 可以集成到任何现有的现代 JavaScript 项目中，也可以从头创建一个全新的 Capacitor 项目。

请按照[安装指南](/main/getting-started/installation.md)开始构建您的应用。
