# Capacitor：跨平台原生运行时，适用于 Web 应用

Capacitor 是一个跨平台原生运行时，让构建能够在 iOS、Android 和 Web 上原生运行的现代 Web 应用变得简单。作为混合应用的下一个演进，Capacitor 创建了 **Web 原生应用**，为希望以 Web 为先同时在不牺牲对原生 SDK 完全访问权限的团队提供了现代原生容器方案。

![Capacitor Logo 和背景](/img/v3/docs/capacitor-index.png)

## 介绍

Capacitor 提供了一套一致且以 Web 为中心的 API，使应用能够尽可能贴近 Web 标准，同时在支持它们的平台上访问丰富的原生设备功能。通过适用于 iOS 的 Swift、Android 的 Java 以及 Web 的 JavaScript 的简单 Plugin API，添加原生功能变得轻而易举。

Capacitor 是 [Apache Cordova](https://cordova.apache.org/) 和 [Adobe PhoneGap](https://phonegap.com/) 的精神继承者，并从其他流行的跨平台工具（如 [React Native](http://facebook.github.io/react-native/) 和 [Turbolinks](https://github.com/turbolinks/turbolinks)）中汲取灵感，但其重点完全在于让现代 Web 应用能够轻松地在所有主要平台上运行。Capacitor 向后兼容许多现有的 [Cordova 插件](https://cordova.apache.org/plugins/)。

## 开始使用

开始使用 Capacitor 很简单！Capacitor 可以集成到任何现有的现代 JavaScript 项目中，也可以从头创建一个全新的 Capacitor 项目。

请遵循 [安装指南](/main/getting-started/installation.md) 开始构建您的应用。
