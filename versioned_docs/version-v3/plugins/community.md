# Capacitor 社区插件

社区已构建了众多 Capacitor 插件，可为你的应用增添各种功能。

有多种方式可以找到社区插件：你可以在网上或 npm 上搜索已发布的插件，也可以浏览官方的 [Capacitor 社区](https://github.com/capacitor-community) GitHub 组织和 [npm 命名空间](https://npmjs.com/~capacitor-community)，那里收录了一系列由社区维护的优质 Capacitor 插件。

## Capacitor 社区

官方的 [Capacitor 社区](https://github.com/capacitor-community) GitHub 组织和 [npm 命名空间](https://npmjs.com/~capacitor-community) 是一项新的举措，旨在精选最佳的 Capacitor 社区维护插件，并提供一致的项目和代码规范。Capacitor 核心团队会协调相关工作，但不会正式维护该组织中的任何插件。

[探索 Capacitor 社区 &#8250;](https://github.com/capacitor-community)

## Cordova 生态系统

Capacitor 支持大多数 Cordova 插件，因此开发者可以在他们的 Capacitor 应用中使用[数百个现有的 Cordova 插件](https://cordova.apache.org/plugins/)。虽然某些 Cordova 插件[与 Capacitor 不兼容](/plugins/cordova.md#known-incompatible-plugins)，但大多数是兼容的，所以如果没有现成的 Capacitor 专用插件，值得尝试一下。

你也可以查看 [Ionic Native 社区](https://ionicframework.com/docs/v3/native/community) 插件，这里主要列出了 Cordova 插件，并提供了便捷的封装器，使它们更容易在你的应用中使用。你可以随意使用 Ionic Native 社区封装器，或者仅仅将该列表作为优质 Cordova 插件的参考。
