# Capacitor 企业版

Capacitor 为关键任务的企业应用提供强大的原生安全、认证和数据管理解决方案及插件。

此外，团队还可以获得深受客户喜爱的支持和可选咨询服务，以确保你的下一个重要应用项目成功并持续保持成功，无论移动生态系统如何变化。

[Capacitor 企业版产品](/plugins/enterprise.md) 围绕一套强大的原生 Capacitor 插件展开：[Auth Connect](https://ionicframework.com/enterprise/auth-connect) 用于安全认证，[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 用于加密会话和用户身份管理，以及 [Offline Storage](https://ionicframework.com/enterprise/offline-storage) 用于高性能加密 SQL 数据存储。

每个解决方案都能与在 Capacitor 中运行的任何现代 Web 技术栈良好配合，并且不需要 Ionic Framework。

## 企业认证

借助 [Auth Connect](https://ionicframework.com/enterprise/auth-connect) 和 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，可以使用移动安全最佳实践（如生物识别认证和加密会话存储）连接到任何 OAuth 或 OpenID 提供商（Auth0、Ping、AWS Cognito、Azure Active Directory 等）。

Auth Connect 为与众多企业和消费者身份提供商的认证提供了一个简单、可移植的 API，仅需几行 JavaScript 代码即可实现复杂的原生认证流程。

结合 Identity Vault（见下文），Capacitor 应用可以实现最强大、最安全且最正确的企业认证流程。

## 安全身份管理

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 与 Auth Connect 协同工作，保护敏感的用户和身份数据（如认证令牌），并通过生物识别认证提供支持，兼容最广泛的 iOS 和 Android 设备。Identity Vault 超越了典型的指纹或面部扫描插件，可与每个平台上的安全 enclave 和加密服务配合使用。这是必要的，因为大多数社区插件仅进行指纹或面部身份扫描，无法提供足够的安全性来防止越狱或其他利用手段。

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 也可以独立使用，并与任何现有认证流程集成，以提供高级会话安全性。

## 离线与加密数据存储

借助 [Offline Storage](https://ionicframework.com/enterprise/offline-storage)，Capacitor 开发者可以使用高性能、加密的 SQL 数据存储系统，在 iOS 和 Android 上构建功能强大且安全的支持离线的应用。

结合 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，开发者可以安全地管理数据加密密钥，并实现设备上的数据管理最佳实践，而无需编写任何原生代码。
