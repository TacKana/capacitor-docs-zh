# 使用插件

Web View 和原生应用通过 Capacitor 或 Cordova 插件进行通信。插件为您的 Web 应用提供原生 API，例如相机、地理位置和文件系统访问。

## Capacitor 插件

Capacitor 团队维护了 [一组常用 API 的 Capacitor 插件](/plugins/official.md)。此外，还有大量来自 [Capacitor 社区](https://github.com/capacitor-community/) 的 Capacitor 插件可供使用。

[了解更多关于 Capacitor 插件的信息 &#8250;](/plugins.mdx)

## Cordova 插件

虽然优先推荐使用 Capacitor 插件，但 Cordova 插件也是一个可选方案。Capacitor 与大多数 Cordova 插件兼容，但在安装时可能需要额外的步骤。

> 如果您因为找不到合适的 Capacitor 插件而使用 Cordova 插件，是否可以考虑 [为 Capacitor 社区创建一个提案](https://github.com/capacitor-community/proposals/)？

[了解更多关于在 Capacitor 应用中使用 Cordova 插件的信息 &#8250;](/plugins/cordova.md)
