# 构建 Ionic 相册应用

**支持平台**：网页、iOS、Android

Capacitor 可以轻松构建同时原生运行于 iOS、Android、桌面和网页端的应用。本教程将带您完成一个全平台兼容的相册应用开发。

> 本教程最新版本已迁移至 Ionic 官方文档站。访问 [最新版教程](https://ionicframework.com/docs/v3/intro/next) 可查看 Angular、React 和 Vue 的实现方案。
