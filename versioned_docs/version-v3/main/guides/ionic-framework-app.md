# 构建 Ionic 照片库应用

**支持平台**: Web, iOS, Android

Capacitor 让构建能够在 iOS、Android、桌面端和 Web 上原生运行的 Web 应用变得简单。在本指南中，我们将构建一个适用于所有平台的完整照片库应用。

> 本指南现在由 Ionic 文档维护。[最新版本](https://ionicframework.com/docs/v3/intro/next) 包含针对 Angular、React 和 Vue 的教程。
