# 构建 Ionic 相册应用

**平台：** Web、iOS、Android

Capacitor 使得构建在 iOS、Android、桌面和 Web 上原生运行的 Web 应用变得容易。在本指南中，我们将构建一个在所有平台上都能运行的完整相册应用。

> 本指南现在在 Ionic 文档中维护。[最新版本](https://ionicframework.com/docs/v3/intro/next)包含 Angular、React 和 Vue 的教程。
